# Premium To-Do Application

A minimalist, high-performance, and responsive To-Do application built using **Vanilla TypeScript** and bundled with **Vite**. This project is designed with a modern user interface, featuring clean typography, soft shadows, interactive micro-interactions, and full CRUD (Create, Read, Update, Delete) capabilities.

## Live Demo
🚀 **[View Live Project on Vercel](https://todo-app-delta-orpin-15.vercel.app/))** 
---

## Features
- **Create:** Add new tasks quickly by clicking the "Add Task" button or pressing the `Enter` key.
- **Read:** Clear and organized dashboard layout showing all tasks dynamically.
- **Update:** Easily modify existing tasks using an intuitive inline prompt system.
- **Delete:** Remove tasks instantly with custom-styled action buttons.
- **Modern UI:** Responsive glass-morphism style built entirely with native CSS variables, containing subtle entry animations and interactive hover effects.
- **Type Safety:** Built on TypeScript to ensure robust DOM manipulation and bug-free logic.

---

## Tech Stack
- **Bundler / Build Tool:** [Vite](https://vitejs.dev/) (For ultra-fast development and optimized production builds)
- **Language:** TypeScript (Vanilla)
- **Styling:** Modern Native CSS3 (Using CSS variables and custom layout systems)
- **Deployment:** [Vercel](https://vercel.com/)
